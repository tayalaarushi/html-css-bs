Introduction


In this assignment you'll learn how to turn any design into responsive pages and to use Bootstrap to make responsive websites like some link

Assignment


Note: Some html and css knowledge is assumed (though not absolutely necessary since everything is done on video) Familiarity with a text editor or IDE of choice is assumed Browser usage is assumed (preferably google chrome)

Description
  1. This course is a practical course for learning how to use html, css and bootstrap to turn designs into fully functional responsive pages
  2. This course is intended for anyone with little html and css experience who wants to see how an actual project is done from scratch.
  3. Even though some html and css knowledge is assumed, though not absolutely necessary since everything is done on video, bootstrap knowledge is not required or assumed as there is a bootstrap crash course included within the course.
  4. Also, you do not need any design software, as I will be using an already made website as a design reference to re-create it again from scratch.
  5. This is done for the reason that not everyone has the same design software installed, and it does not make a difference if the design is referenced in a finished page like in this course or actual design such as photoshop, illustrator or sketch.
  6. The only difference is that you need some browser extensions to check colors, widths, heights and spaces between elements.
  7. Since I do not use design from software such as photoshop, illustrator or sketch, I will be using some chrome extensions that I generally use when doing front-end work, and you can check these out in a free video as well.
  8. Once you have completed the course you will be able to turn designs into into fully functional responsive pages.
